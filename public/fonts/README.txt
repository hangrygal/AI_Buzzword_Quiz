UNCUT SANS FONTBESTANDEN
========================

Leg hier je Uncut Sans webfonts neer (.woff2). Verwachte bestandsnamen
(gebruikt door de @font-face-regels in src/index.css):

    UncutSans-Book.woff2        (gewicht 400 — bodytekst)
    UncutSans-Semibold.woff2    (gewicht 600 — headings/sublines)

Zodra deze bestanden hier staan, gebruikt de app automatisch Uncut Sans.
Ontbreken ze? Dan valt de app netjes terug op Inter / systeem-grotesque,
zonder dat je code hoeft aan te passen.

Het logo hoort in ../logo/  (mark-black.svg / mark-white.svg vervangen).
